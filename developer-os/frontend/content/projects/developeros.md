# DeveloperOS

A browser-based **Software Engineer Operating System** — a portfolio reimagined
as a real, explorable desktop OS.

## Overview

Instead of a traditional portfolio page, DeveloperOS presents projects,
experience, and engineering thinking through windows, a taskbar, a start menu,
and desktop apps (About.exe, Projects.exe, Experience.exe, AIAssistant.exe, and
more).

## Tech Stack

- **Framework:** Next.js (App Router), React, TypeScript
- **Styling:** Tailwind CSS
- **State:** Zustand
- **Animation:** Framer Motion
- **3D:** React Three Fiber / Three.js (ambient desktop background)
- **AI:** OpenAI API with a markdown knowledge base

## Key Features

- Full window manager (open, focus, drag, minimize, maximize, close)
- Windows 11-style taskbar and start menu
- 3D ambient desktop background with performance guardrails
- AI Assistant that answers questions from a markdown knowledge base

## Status

In active development.
