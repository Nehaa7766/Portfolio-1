# Hospital Management System

A full stack hospital management solution with appointment booking, patient
management, billing, and reporting.

## Overview

The system streamlines day-to-day hospital operations: scheduling
appointments, maintaining patient records, generating bills, and producing
operational reports.

## Tech Stack

- **Frontend:** React, Tailwind CSS
- **Backend:** Spring Boot (REST APIs)
- **Database:** MySQL

## Key Features

- Appointment booking and conflict detection
- Patient management (records, history)
- Billing and invoicing
- Reporting and dashboards
- Role-based access (admin, doctor, staff)
