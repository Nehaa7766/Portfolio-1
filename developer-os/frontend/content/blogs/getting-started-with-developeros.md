# Building DeveloperOS — A Portfolio as an Operating System

Most portfolios are a single scrolling page. I wanted something that *shows*
how I think as an engineer, not just lists what I've done.

DeveloperOS reframes the portfolio as a desktop operating system: each piece of
work is an app you can open, drag, and explore. It demonstrates state
management, a window manager, performance-conscious 3D, and an AI assistant —
all the things I care about as a developer.
